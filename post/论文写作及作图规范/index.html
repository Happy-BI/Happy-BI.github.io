论文写作及作图规范

论文写作及作图规范 🎓

Huan BI (毕欢)
2017-07-12 1 min read 教程

写作前构思好文章的创新点在哪里。

  • 标题：要突出文章最亮的点，标题应反复琢磨。一个好的标题可为文章润色不少。
  • 作者：人名在前，姓在后，通讯作者加*号。
  • 单位：要详实，准确，第一单位放在最前。

正文

  • 小四字体；Times New Roman；双倍行距；标点符号前加空格（% 例外）；正文中x要斜体；离子符号要插入横杠，如I−，不能直接打“减号”；摄氏度符号要插入，不能用输入法直接打出；
  • 例如：FA(1-x)MAxPbI3，x = 0.1 等，比例用小写x，大写X代表卤素；1-x要括号，且下标；所有符号在英文状态下输入，且x = 0.1，等号前后有空格；挨着()的不要空格，如（x = 0, 0.1）。
  • 注意：（）和()两种括号，所有符号保证为英文格式下输入。
  • x = 0 – 0.1，表示一个范围时，连字符左右空格；表示一个整体时不用空格，例如DMSO-d6；organic-inorganic hybrid perovskite。

参考文献

  • 参考文献要按逻辑及出现顺序依次编号；缩写一定要在第一次用的时候给出定义，例如power conversion efficiency (PCE)，术语全称要小写。

图片

  • 图要求：上下对齐，简洁美观，坐标名称32号字体加粗，坐标轴厚度及数据线都用三号线，轴上的数字28号字体加粗；配色美观；文章中图片的引用原则是要标明引文出处，不能修改比例，可以调节大小；可以利用PS将不同文献中的图重新编号，当然编号也要在整篇文章中使用同一格式。章中提到Figure N要统一编号并且最好加粗，这样找起来方便。在标明图片来源时，一般加在图例的最后一句上，并标明Copyright+年份+出版社，如下图所示。

画图过程的逻辑性

  1. 位置：
    • 坐标：先相似的图保证骨架一样，如横坐标范围，纵坐标范围，横坐标名称，纵坐标名称位置，可以先调整好一个之后，再用模板。
    • 数据画图：调整坐标范围以及数据的纵坐标统一修改，保证图在一定黄金位置，不占满，也不留很多空位，给图例等留下位置。
    • 在Origin中title Wavelength / nm 分隔号前后需要空格。
    • 所有图都用 / 来表示坐标轴单位。
  2. 颜色：
    • 数据线颜色配色要统一，好看且清晰。
  3. 输出格式：
    • Origin输出：TIF格式，300DPI，保存后用画图打开，另存为PNG即可。
    • 当图处于画布（背景白板）太下方时，输出不全，需要选中该图片，size/speed 中TOP变小。
  4. PS处理几幅图：
    1. 新建画布，宽度16cm（输出为全列，8.5cm输出为半列），高度不限，600DPI。然后打开需要画的图，把其他图移到新建画布中，并用control+T固定长宽比，缩放；
    2. PS加ABCD（Bold 浑厚10点的A，对应Title28，比较好看），以刻度尺为参考系，因为该图是可以移动的。加入A的位置，以及剪裁的位置都是固定的；
    3. 查看像素，16cm 宽以及600DPI，而后保存为PSD程序，可供打开，最后再保存TIF，无压缩即可。
  5. Word插图：
    • 在Word里选项设置插入图片不压缩，220ppi，然后复制黏贴即可。

参考文献格式

  • 刊物名称斜体不加粗，缩写正确；年份加粗；卷号斜体；页码不加粗不斜体；格式统一。
  • 在提交文稿时要删去待链接的参考文献而替换成文本格式。
